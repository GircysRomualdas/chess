# Chess

Chess Game built with Python and Pygame.

---

## Requirements

- Python 3

---

## Installation

1. Clone the Repository

2. Create virtual environment and activate.
```bash
python3 -m venv env
source env/bin/activate
```

3. Install dependencies
```bash
pip install -r requirements.txt
```

---

## Run the Game

```bash
python3 main.py
```
